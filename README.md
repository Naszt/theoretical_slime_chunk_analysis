# 史莱姆区块课题

我们知道，在 Minecraft 中史莱姆是一种很重要的生物，查看 wiki 可以发现：
> 史莱姆生成在40层以下的特定的“史莱姆区块”里（不限亮度）。

于是想要制造高效率的史莱姆农场就需要一片较大的多联史莱姆区块。  
但是由于现版本渗浆药水的存在，该任务就变成了一个纯粹的理论课题。

## 文件管理

### 文件夹
`2n` 文件夹是原本的找 $2\times n$ 的算法，  
不过是让 $B$ 很大，$c\in[0,2^{32}-1]$，  
但由于 $H$ 函数的奇妙周期性，其实跑的并不算慢。

`dick` 文件夹是找 $6$ 长 dick 的代码，修改自 `2n` 代码。

`2n_single` 文件夹是考虑 `B=1` 的代码，  
它的思路是扩展向上下找长条，这样结果一定包含 $z$ 坐标为 $Z$ 的区块。

### 工具
`chuckbase.js` 是 [chunkbase](https://www.chunkbase.com/apps/seed-map) 的油猴脚本，可以批量查看种子。  
`2n` 中的 `draw.cpp` 也可以批量查看种子，是跑在本地的。
